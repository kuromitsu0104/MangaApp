# ChatApp

Vue3 + TypeScript + RailsAPI 環境での開発練習用

# Project setup

ソースコードを取得

```
$ git clone https://github.com/kuromitsu0104/ChatApp.git
```

パッケージをインストールして、ローカル環境に node_modules を生成

```
$ cd frontend
$ npm install
```

docker-compose コマンドでビルド

```
$ docker-compose build
```

docker-compose コマンドで起動

```
$ docker-compose up
```
